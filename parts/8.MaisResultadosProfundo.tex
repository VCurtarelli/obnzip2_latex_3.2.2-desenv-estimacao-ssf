\section{Simulações com perfis até 5000m}
\label{sec:mais_simulacoes_profundo}

As simulações previamente apresentadas mostravam os resultados da estimação do SSF para um ambiente até $1500\si{\meter}$ de profundidade com 31 células. Enquanto um ambiente raso é de certo interesse, seu comportamento não necessariamente reflete de maneira plena todas as situações válidas. Para tal, agora utilizaremos um recorte do WOA com $4900\si{\meter}$ de profundidade com 50 células, utilizando dados anuais do banco de dados. Foi necessário alterar o local tal que houvesse dados até tal profundidade, agora centrado em $36.375\dg$S, $38.375\dg$W. O SSF utilizado, juntamente com os perfis 1D (de cada coluna) para esse ambiente encontram-se nas \cref{fig:sec8:discretized_ssf_profiles,fig:sec4:discretized_ssf_true}.

Comparando com o perfil utilizado até então da \cref{fig:sec4:discretized_ssf_true,fig:sec4:discretized_ssf_profiles}, nota-se uma homogeneidade horizontal muito maior entre os perfis, mas uma variação muito maior e menos uniforme com a profundidade.

\input{figures/8.a.ssf_discretizado}

As novas condições de distribuição dos dispositivos são tal como mostrado na \cref{fig:sec8:dispositivos}. Por consistência, foi mantido que os receptores somente poderiam viajar até $500\si{\meter}$ no caso 1, dada que essa seria uma restrição externa dos dispositivos. Em razão do aumento de mais de 3x de profundidade e de 60\% do número de células, decidiu-se aumentar a quantidade de dispositivos na porção vertical das configurações do grupo 3 de dois (de cada) para quatro (de cada). As configurações do grupo 2 são idênticas às utilizadas na \cref{sec:mais_simulacoes_raso}, re-escaladas para a nova profundidade.
